# esphome_components

Misc. unofficial ESPHome components.

## Usage

```yaml
external_components:
  - source:
      type: git
      url: https://github.com/fightforlife/esphome-components
    components: [ ethernet ]
```

## Components

### [ethernet](components/ethernet)

Component to support W5500 on ESP-IDF and Arduino as a ESPHome component.
